üíª Code Template (Python)

def sliding_window_template(arr, k):
    window_start = 0
    window_sum = 0
    max_sum = float('-inf')
    
    for window_end in range(len(arr)):
        window_sum += arr[window_end]  # Add the next element
        
        # Slide the window if we've hit the window size
        if window_end >= k - 1:
            max_sum = max(max_sum, window_sum)
            window_sum -= arr[window_start]  # Remove element going out
            window_start += 1  # Slide window ahead
    
    return max_sum

Two pointers excel at sorted array problems and palindrome checks. Use when you need to:

  • Find pairs/triplets summing to target

  • Check if string is palindrome
  • Merge sorted arrays

üíª Code Template (JavaScript)

// Two pointers for sorted array
function twoPointersTemplate(arr, target) {
    let left = 0;
    let right = arr.length - 1;
    
    while (left < right) {
        const sum = arr[left] + arr[right];
        
        if (sum === target) {
            return [left, right];
        } else if (sum < target) {
            left++; // Need larger sum
        } else {
            right--; // Need smaller sum
        }
    }
    
    return [-1, -1]; // No pair found
}

// For SDET: Useful for comparing arrays or finding differences

from collections import deque

def bfs_template(root):
    if not root:
        return []
    
    result = []
    queue = deque([root])
    
    while queue:
        level_size = len(queue)
        current_level = []
        
        for _ in range(level_size):
            node = queue.popleft()
            current_level.append(node.val)
            
            # Add child nodes to queue
            if node.left:
                queue.append(node.left)
            if node.right:
                queue.append(node.right)
        
        result.append(current_level)
    
    return result

Medium

How would you find the longest substring without repeating characters?

Time: 25 min

üéØ Expected Approach

Use a sliding window with a hash map to track character positions. Expand the window until you find a duplicate, then shrink from the left until the duplicate is removed.

Pattern: Sliding Window with Hashing

Time: O(n), Space: O(min(n, m)) where m is character set size

üíª Solution (Python)

def length_of_longest_substring(s: str) -> int:
    char_index = {}  # Stores last index of each character
    left = 0
    max_length = 0
    
    for right in range(len(s)):
        # If character exists in window, move left pointer
        if s[right] in char_index and char_index[s[right]] >= left:
            left = char_index[s[right]] + 1
        
        # Update character's last seen index
        char_index[s[right]] = right
        
        # Update max length
        max_length = max(max_length, right - left + 1)
    
    return max_length

üîß SDET Application

This pattern is useful for validating unique identifiers, checking for duplicate test case names, or analyzing log sequences for patterns in test automation.
