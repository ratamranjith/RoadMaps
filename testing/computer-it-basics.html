1. Introduction to Computers

Understanding the fundamental concepts of computing

What is a Computer? - A Detailed Definition

A computer is an electronic device that processes data according to a set of instructions called a program. It can:

  • Accept input (data or instructions)
  • Process the input according to specified rules
  • Produce output (information)
  • Store information for future use

The Basic Computer Model:

Von Neumann Architecture
üèõÔ∏è

Input ‚Üí CPU ‚Üí Memory ‚Üí Output

The fundamental architecture of modern computers

All modern computers follow this basic architecture with five essential components:

  1. Input Unit: Receives data and instructions
  2. Memory Unit: Stores data and instructions
  3. Arithmetic Logic Unit (ALU): Performs calculations
  4. Control Unit (CU): Manages instruction execution
  5. Output Unit: Presents processed information

Characteristics of Computers

Speed

Modern computers perform millions to billions of operations per second (measured in GHz)

Accuracy

Computers process data with near-perfect accuracy when hardware and software function correctly

Storage Capacity

Can store vast amounts of data (terabytes to petabytes) with various storage hierarchies

Diligence

Can perform repetitive tasks without fatigue, maintaining consistent performance

Versatility

Can perform different types of tasks by loading different programs

Automation

Can execute tasks automatically once programmed, requiring minimal human intervention

Central Processing Unit (CPU) - The Brain of Computer

CPU Components:
  • ALU
    Arithmetic Logic Unit

    Performs mathematical operations (add, subtract, multiply, divide) and logical operations (AND, OR, NOT)

  • CU
    Control Unit

    Directs operation of processor, manages fetch-decode-execute cycle, controls data flow between components

  • REG
    Registers

    Small, fast storage locations within CPU (accumulator, program counter, instruction register, memory address register)

  • CACHE
    Cache Memory

    Small, fast memory between CPU and RAM (L1, L2, L3 caches)

CPU Specifications:

Measured in GHz (billions of cycles per second). Higher clock speed = faster processing, but also more heat/power consumption.

Example: 3.5 GHz = 3.5 billion cycles/second

Cores: Independent processing units within a CPU. Threads: Virtual cores managed by OS.

Dual-core, Quad-core, Octa-core, etc.

CISC (Complex Instruction Set Computer) vs RISC (Reduced Instruction Set Computer)

x86 (Intel/AMD), ARM (mobile/embedded), RISC-V (open standard)

The Fetch-Decode-Execute Cycle:

  1. Fetch: CPU retrieves instruction from memory (RAM)
  2. Decode: Control unit interprets the instruction
  3. Execute: ALU performs the operation
  4. Store: Results are written back to memory
  5. Repeat: Program counter increments, next instruction

This cycle happens billions of times per second in modern CPUs.

5. Networking Architecture

Connecting computers and devices for communication

Network Fundamentals

What is a Network?

A network is a collection of computers, servers, mainframes, network devices, peripherals, or other devices connected to one another to allow sharing of data and resources.

LAN (Local Area Network)

Covers small geographical area (home, office, building). Uses Ethernet or Wi-Fi.

High Speed Low Latency
WAN (Wide Area Network)

Spans large geographical area (cities, countries). The Internet is the largest WAN.

Global Reach Slower Speed
MAN (Metropolitan Area Network)

Covers city or campus. Larger than LAN but smaller than WAN.

City-wide Medium Speed
